# Инструкция по резервному копированию БД

## Как создать бэкап
1. Открыть командную строку (cmd)
2. Перейти в папку проекта
3. Выполнить команду:
"рассположение сервера mysql bin" -u root -p uyut_db > backup.sql
4. Ввести пароль от MySQL

## Как восстановить из бэкапа
1. Открыть командную строку (cmd)
2. Перейти в папку проекта
3. Выполнить команду:
"расположение сервера mysql bin" -u root -p < backup.sql
4. Ввести пароль от MySQL

## Примечания
- Файл бэкапа сохраняется в текущей папке
